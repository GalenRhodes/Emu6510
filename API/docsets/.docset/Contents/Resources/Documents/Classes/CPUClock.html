CPUClock

public class CPUClock

A timer that calls the provided closure once every timer tick.

  • freqency

    The frequency of the clock in Hz. Due to CPU speed limitations there exists an effective high frequency. For example, on a 3.2GHz MacBookPro+Retna with 16GB of RAM it takes about 30-40ns to get the system monotonic raw time. This means that, not counting any other overhead like actually doing anything but directly returning from the closure, the effective max frequency is about 20-22 MHz.
